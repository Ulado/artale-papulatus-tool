# artale-papulatus-tool
tool for papulatus difficult mode

## 各功能設定說明
- 點擊左側選單中的 連結遊戲畫面 按鈕
- 調整 即時監控畫面 中的 偵測框 位置與大小
  - 血量偵測框：框住拉圖斯血量 數字與趴數 即可
  - 對話偵測框：框住對話框中最後一行 新對話會出現的地方即可
  - 小地圖偵測框：框住小地圖即可
- 調整 小地圖預覽 中的 標記線 位置 (參考左側安全點位置調整)
- 根據 小地圖預覽預覽 畫面中檢查是否顯示綠色準心(玩家位置)
    - 有顯示綠色準心代表 RGB 設定正確
    - 沒有顯示綠色準心代表 RGB 設定需要調整, 請調整 RGB 設定 中的 R、G、B 數值與誤差容許值，直到小地圖預覽中的準心顯示出來為止 
- 點擊儲存設定按鈕 

## 各功能用途說明
- 血量偵測
  - 不同 HP % 會有不同攻擊機制
  - 會顯示文案提醒玩家目前有哪些攻擊機制
  - 手動調整血條滑桿可以看到不同文案說明
- 對話偵測
  - 當關鍵字 "注意你的腳下" 出現時會自動倒數 (預設 70s, 下次黑水平均出現的時間間隔)
  - 手動點擊重置按鈕可以停止倒數與重置倒數秒數
  - 手動點擊倒數計時可以馬上開始倒數
- 小地圖偵測
  - 依照 RGB 顏色判斷 小地圖中玩家的位置
  - 當玩家位置靠近標記線時右上角會顯示 SAFE 狀態 (代表站在安全位置上, 不會被紅鬧鐘炸到)
